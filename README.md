# My Data Science Capstone Project

## SpaceX Launch Success Data Analysis & Prediction

This repository contains the final capstone project for the **IBM Data Science Professional Certificate**.

### Project Objective
Analyze SpaceX Falcon 9 launch data to identify factors that affect first-stage landing success and build machine learning models to predict landing outcomes.

### Repository Structure
- `notebooks/` – Jupyter notebooks covering data collection, data wrangling, EDA, interactive visualization, and predictive analysis
- `presentation/` – Final project presentation (PDF)

### Technologies Used
Python, Pandas, NumPy, SQL, Folium, Plotly Dash, Scikit-learn
